# Project: copytf

This program can be used to copying a text file (ASCII Code).

# Getting Started

## Prerequisites

System: GNU/Linux

Compiler: GCC(All Versions)

C Standard Library: The GNU C Library(glibc)(All Versions)

## Installation

GNU/Linux:

git clone https://github.com/rymeng/copytf.git

cd copytf

mkdir bin

mkdir obj

make
